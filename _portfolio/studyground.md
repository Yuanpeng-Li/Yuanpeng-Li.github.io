[StudyGround](https://github.com/Yuanpeng-Li/StudyGround) is a Claude Code plugin for AI-assisted self-study. Claude Code writes the lessons; you drive the pace — read rendered lessons in a local web reader, talk to a course-aware tutor, run Python in the browser via Pyodide, and jump into VSCode for hands-on exercises.

Each course is a folder with its own curriculum, lessons, materials, threads, and progress. Lessons carry live markers: inline questions become **Ask** buttons, runnable Python blocks execute in the browser, and exercise blocks scaffold a folder and deep-link into VSCode. Highlight a passage to start a side chat that folds back into the lesson when you save it, and drop PDFs or notes into a course to ground newly generated lessons in your own materials.
